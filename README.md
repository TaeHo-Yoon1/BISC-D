# 드보락 키보드 타자연습 & 코딩 연습 프로그램

드보락 키보드 레이아웃과 프로그래밍 코드를 연습할 수 있는 종합 타자연습 프로그램입니다.

## 주요 기능

### 일반 타자연습
- **드보락 키보드 레이아웃 표시**: 화면에서 드보락 키보드 레이아웃을 확인할 수 있습니다
- **실시간 통계**: 타이핑 속도(WPM), 정확도, 진행률을 실시간으로 확인
- **다양한 연습 모드**: 
  - 기본 연습 텍스트
  - 난이도별 연습 (초급/중급/고급)
  - 사용자 정의 텍스트 파일 로드

### 코딩 연습 모드 🚀
- **다양한 프로그래밍 언어 지원**: Python, Java, JavaScript, C++, React
- **난이도별 코드 템플릿**: Basic, Intermediate, Advanced
- **문법 하이라이팅**: 키워드, 문자열, 주석 등을 색상으로 구분
- **코드 템플릿 선택**: 언어와 난이도를 선택하여 원하는 코드 연습
- **실제 코드 연습**: 실제 프로그래밍 코드를 따라 치며 코딩 타이핑 속도 향상

### 공통 기능
- **통계 저장**: 일반 타자연습과 코딩 연습 기록을 분리하여 저장
- **상세한 통계 분석**: 언어별, 모드별 통계 제공
- **사용자 친화적 GUI**: 직관적이고 깔끔한 인터페이스

## 시스템 요구사항

- Windows 10 이상
- Python 3.7 이상 (소스코드 실행 시)

## 설치 및 실행

### 방법 1: 실행 파일 사용 (권장)

1. `dist` 폴더에서 `DvorakTypingTrainer.exe` 파일을 다운로드
2. 더블클릭하여 실행

### 방법 2: 소스코드에서 실행

1. Python이 설치되어 있는지 확인
2. 필요한 패키지 설치:
   ```bash
   pip install -r requirements.txt
   ```
3. 프로그램 실행:
   ```bash
   python main.py
   ```

### 방법 3: 실행 파일 직접 빌드

#### Windows에서:
```bash
build_exe.bat
```

#### Linux/Mac에서:
```bash
chmod +x build_exe.sh
./build_exe.sh
```

## 사용법

### 일반 타자연습 모드
1. **모드 선택**: "일반 타자연습" 라디오 버튼 선택
2. **새 연습 시작**: 기본 연습 텍스트로 새로운 연습을 시작합니다
3. **파일에서 로드**: 사용자가 준비한 텍스트 파일을 불러와 연습할 수 있습니다
4. **난이도 선택**: 초급/중급/고급 난이도 중 선택하여 연습할 수 있습니다

### 코딩 연습 모드
1. **모드 선택**: "코딩 연습" 라디오 버튼 선택
2. **언어 선택**: Python, Java, JavaScript, C++, React 중 선택
3. **난이도 선택**: Basic, Intermediate, Advanced 중 선택
4. **새 연습 시작**: 선택한 언어와 난이도의 랜덤 코드 템플릿으로 연습 시작
5. **코드 선택**: '코드 선택' 버튼으로 특정 코드 템플릿을 선택하여 연습

### 공통 사용법
1. **타이핑**: 화면에 표시된 텍스트/코드를 입력창에 따라 타이핑합니다
2. **실시간 통계**: 실시간으로 타이핑 속도와 정확도를 확인할 수 있습니다
3. **기록 보기**: '통계 보기' 버튼으로 과거 연습 기록을 확인할 수 있습니다
4. **모드 전환**: 언제든지 일반 타자연습과 코딩 연습 모드 간 전환 가능

## 드보락 키보드 레이아웃

프로그램 상단에 드보락 키보드 레이아웃이 표시됩니다:

```
` 1 2 3 4 5 6 7 8 9 0 [ ] \
  q w e . p y f g c r l
   a o e u i d h t n s -
    ; q j k x b m w v z
```

## 통계 기능

### 기본 통계
- **WPM (Words Per Minute)**: 분당 단어 수로 타이핑 속도 측정
- **정확도**: 올바르게 입력한 문자의 비율
- **진행률**: 전체 텍스트 중 입력 완료한 비율

### 상세 통계 분석
- **모드별 통계**: 일반 타자연습과 코딩 연습을 분리하여 통계 제공
- **언어별 통계**: 코딩 연습에서 프로그래밍 언어별 성과 분석
- **연습 기록**: 모든 연습 세션의 기록을 JSON 파일로 저장
- **최근 기록**: 최근 10회 연습 기록을 상세히 표시

## 파일 구조

```
BISC-D/
├── main.py                 # 메인 프로그램 파일
├── coding_templates.json   # 프로그래밍 언어별 코드 템플릿
├── practice_texts.txt      # 연습용 텍스트 파일
├── requirements.txt        # 필요한 Python 패키지 목록
├── build_exe.bat          # Windows용 빌드 스크립트
├── build_exe.sh           # Linux/Mac용 빌드 스크립트
├── README.md              # 사용 설명서
├── typing_stats.json      # 통계 데이터 (자동 생성)
└── dist/                  # 빌드된 실행 파일 위치
    └── DvorakTypingTrainer.exe
```

## 문제 해결

### 실행 파일이 실행되지 않는 경우
- Windows Defender나 백신 프로그램에서 차단하는지 확인
- 관리자 권한으로 실행해보기
- 다른 컴퓨터에서 실행해보기

### Python 환경에서 실행 시 오류
- Python 버전이 3.7 이상인지 확인
- 필요한 패키지가 모두 설치되었는지 확인:
  ```bash
  pip install -r requirements.txt
  ```

## GitHub 릴리즈 생성 방법

### 방법 1: 스크립트 사용 (권장)

#### Windows에서:
```bash
create_release.bat v1.0.0 "첫 번째 릴리즈"
```

#### Linux/Mac에서:
```bash
chmod +x create_release.sh
./create_release.sh v1.0.0 "첫 번째 릴리즈"
```

스크립트 실행 후:
1. https://github.com/TaeHo-Yoon1/BISC-D/releases/new 로 이동
2. 'Choose a tag'에서 생성된 태그 선택
3. 릴리즈 제목과 노트 입력 (RELEASE_NOTES_TEMPLATE.md 참고)
4. (선택사항) 빌드된 실행 파일 첨부
5. 'Publish release' 버튼 클릭

### 방법 2: 수동으로 태그 생성

```bash
# 태그 생성
git tag -a v1.0.0 -m "릴리즈 노트"

# 태그 푸시
git push origin v1.0.0
```

그 후 GitHub 웹사이트에서 릴리즈를 생성하세요.
